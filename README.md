## Installation

Using pyrogram for task managing. Python version must be >3.8.

Install libraries:

```
pip install -r requirements.txt
```

Build docker: 

```
docker-compose build
```

Start docker:

```
docker-compose build -d
```

Manual start:

```
cd app && python3.9 app.py
```
